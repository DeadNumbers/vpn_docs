# **Azure**

---

Our azure integrations allow you to import users from your Azure Active Directory, use Azure SAML authentication, set up 2FA and more. We provide instructions to help you get up and running as fast as possible, see the links below and follow the instructions based on what you need. If you need any help let us know at [support@ironwifi.com](mailto:support@ironwifi.com) or schedule a call with us [here](https://calendly.com/ironwifi/15min).



- [Azure AD SAML - Authentication using SAML](https://www.ironwifi.com/azure-ad-saml/)
- [Azure AD OAuth - Authentication using OAuth](https://www.ironwifi.com/azure-ad-oauth/)
- [Azure Active Directory - Manage your wireless users using Azure Active Directory account, enable remote synchronization with your Azure account for users in specific groups](https://www.ironwifi.com/azure-active-directory/)
- [Azure AD Authentication with PEAP-MSCHAPv2](https://www.ironwifi.com/azure-ad-authentication-peap-mschapv2/)
- [Azure 2-Factor Authentication](https://www.ironwifi.com/azure-2-factor-authentication/)
